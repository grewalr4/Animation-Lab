# LaboratoryDemo

This is the main project for LaboratoryDemo. It only implements 
an application class and a dependency injection for the 
laboratory that is passed on to LaboratoryDemoLib that implements
the actual demonstration program. LaboratoryDemoLib is 
automatically pulled from github by CMake rather than
being permanently included in this project.

**Do not modify any part of LaboratoryDemo!**
